O que é HTML Semântico?

HTML Semântico refere-se ao uso de tags HTML que transmitem o significado do conteúdo nelas contido, tornando a estrutura da página mais compreensível tanto para humanos quanto para máquinas (como mecanismos de busca e leitores de tela).

Por que usar HTML Semântico?

  • Melhora a acessibilidade para pessoas com deficiência.
  • Facilita o entendimento do conteúdo pelos mecanismos de busca (SEO).
  • Organiza melhor o código, tornando-o mais fácil de manter.

Exemplo de HTML Não Semântico

<div id="header">Cabeçalho</div>
<div id="menu">Menu</div>
<div id="content">Conteúdo principal</div>
<div id="footer">Rodapé</div>

Exemplo de HTML Semântico

<header>Cabeçalho</header>
<nav>Menu</nav>
<main>Conteúdo principal</main>
<footer>Rodapé</footer>

Principais Tags Semânticas

  • <header>: Cabeçalho da página ou seção
  • <nav>: Navegação principal
  • <main>: Conteúdo principal
  • <section>: Seção de conteúdo
  • <article>: Conteúdo independente
  • <aside>: Conteúdo lateral ou complementar
  • <footer>: Rodapé da página ou seção
